Document SSO (Social and Search Optimization)

WPSSO Core includes an elegant "Social and Search Optimization" metabox for editing pages (for both the classic and the new block editor in WordPress 5), allowing you to fine-tune and customize the title, description, image details, video information, etc. that Facebook, Pinterest, Twitter and other social sites use.

The Document SSO metabox shows a different set of options based on the content type (aka Schema type) selected, allowing you to customize the details of articles, events, e-Commerce products, recipes, reviews, etc.

The Document SSO metabox also provides a social share preview, a complete list of meta tags and markup created, and validation buttons for several social / search tools. Complementary add-ons can include place / location features, mobile app options, and much more.
